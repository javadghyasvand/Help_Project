Les paramètres généraux liés au matériel de l'appareil sont ajustés dans la section "Hardware" de la page "Preference".  Dans la section « Fan mode », en sélectionnant l'option « Silent » modifiera la vitesse du « Fan » en fonction de la température des interrupteurs ou radiateurs de l'appareil qui sont affichés sur cette page.  Mais, si l'option "Max.Power" est sélectionnée, le ventilateur de l'appareil fonctionnera à sa puissance maximale en permanence.

En activant l'option "Smart Delay", si la température de l'appareil atteint 55 degrés, dans les pièces telles que "Distance" et "OverCurrent" où les points sont utilisés pour le test, le test se mettra automatiquement en pause et un message disant "Cooling "s'affichera dans la barre inférieure de l'écran de test.  Le test ne reprendra que lorsque la température atteint 45 degrés.  La température maximale de fonctionnement de l'appareil est spécifiée dans la section « Maximum Temperature ».  Si la température de l'appareil dépasse cette valeur, l'appareil s'arrêtera complètement de fonctionner.  Cette valeur peut être changée jusqu'à 70 degrés mais, sauf dans certaines conditions particulières, l'utilisateur ne doit pas régler la température au-dessus de 60 degrés.

Dans la section "Switch Off After Test", l'utilisateur peut régler combien de secondes après la fin du test, les commutateurs de courant et de tension doivent être ouverts.  Ce nombre est fixé à 5 secondes par défaut mais il peut être augmenté jusqu'à 10 secondes. Dans la section « Deviation», les conditions de déclaration d'erreur de l'appareil « Other » peuvent être spécifiées.  Par défaut, il est spécifié dans le logiciel que s'il y a une différence entre la tension réelle de l'appareil et la valeur spécifiée dans le logiciel dépasse 4 volts ou si la différence de courant dépasse 500 mA, l'erreur "Other" doit être indiquée.  Dans la section "Relative", cette valeur est spécifiée en pourcentage de la valeur spécifiée dans le test.  Dans le mode de fonctionnement réel, l'appareil utilise la valeur la plus faible parmi ces deux pour le message d'erreur.

Dans la section "Binary Input Status", il est indiqué quel binaire est connecté et quel binaire déconnecté.

Dans la section "Disable error", une liste des erreurs possibles de l'appareil est disponible et vous pouvez les désactiver.

Cette erreur: "Select All Error Other" s'affiche lorsque l'appareil est incapable de produire le courant et la tension demandés par l'utilisateur.  La cause de cette erreur peut être le « court-circuit » des sorties de tension ou le circuit ouvert des sorties de courant.  En ouvrant la liste déroulante de chaque champ d'erreur, toute erreur peut être désactivée pour un certain nombre de sorties de tension et de courant.

"Error Thermal" : cette erreur s'affiche lorsque la température des capteurs de l'appareil  augmente.  Parfois, le capteur thermique de l'un des commutateurs a un problème, affichant des températures déraisonnables telles que 800 degrés.  Dans ce cas, dans le curseur de cette section, il est possible de désactiver l'erreur de température pour continuer à travailler avec l'appareil.

"Error Over Current Binary 9": le binaire 9 de l'appareil peut mesurer le courant jusqu'à 500mA.  Si le courant d'entrée de ce binaire dépasse cette valeur, cette erreur apparaîtra.

Dans le champ "Times For ignoring Over Voltage Binary", la durée maximum autorisée pour ignorer l'erreur "Over Voltage" du binaire est définie.  Par défaut, il est défini que si le temps de l'erreur "Over Voltage" est inférieur à 100 millisecondes, l'appareil ne montra aucune "Erreur" à ce sujet.

Il convient de noter que, étant donné que "Thermal Error" et "Error Over Current Binary 9" font partie des "Error" dangereuses de l'appareil, leur paramètre de désactivation ne sera pas enregistré par le logiciel et sera désactivé à chaque exécution du logiciel , alors si cette "Error" doit être désactivée, l'utilisateur doit le faire consciemment.  La section "Check RAM" est utilisée pour tester la RAM qui est utilisée dans "AMT".  Dans le champ "Repetition", le nombre de fois qu'une série de "Data" sont écrites dans la RAM et supprimées est définie.  Cette section peut être apliquée en cas d'incertitude sur les fonctions de la RAM de l'appareil.  Étant donné que les opérations d'écriture et de lecture sont effectuées plusieurs fois sur la RAM, éventuellement, s'il y a un problème dans ce processus, le voyant de "error" sur le panneau de l'appareil s'allumera.

Dans la section "Earth", il est possible d'activer ou de désactiver l'erreur "Earth" de l'appareil.  Si l'option "Enable" est cochée, il est nécessaire de brancher le câble "Earth" pour effectuer le test.  Sinon, une erreur "Earth" s'affichera et l'appareil cessera de fonctionner. L'erreur" Earth" peut être désactivée en décochant cela.  Mais, il est essentiel de noter que cela peut mettre la vie en danger pendant le test et comme il est également affiché dans le message, la société Vebko ne sera pas responsable.

Il est possible d'activer ou de désactiver les principaux commutateur matériels de l'appareil dans la section "Switches".  Le commutateur numéro 1 est destiné aux tensions et courants de la sortie du groupe A.  Le commutateur numéro 2 est destiné aux tensions et courants de sortie du groupe B et VDC.  En désactivant chaque interrupteur, toutes les sorties de cet interrupteur seront désactivées.

Dans la section "Combination cable", si l'option "Voltage" est sélectionnée, les sorties du câble Neutrik comprendront trois phases de courant et trois phases de tension.  Mais si l'option "Current" est sélectionnée, la sortie du câble sera composée de deux groupes de courant triphasé qui peuvent être utilisés pour divers tests tels que le test différentiel.

L'option "Open VDC Setting" permet d'activer la tension de sortie DC de l'appareil.  En cliquant sur cette option, la page "VCD Diagrams" s'ouvre.  Sur page, dans le champ " DC Value" de la section "Apply", la valeur DC de sortie de ce port est définie.  Cette valeur peut être réglée jusqu'à un maximum de 212 Volts "DC".  En cliquant sur l'option "Apply", le port "AUX DC" de l'appareil injectera la valeur qui est spécifiée dans le champ "Value DC".  La sortie "AUX DC" peut être désactivée en cliquant sur l'option "Disable DC".  En activant l'option "Record Period" dans le champ suivant, le temps est définie pour l'enregistrement du signal de sortie de ce port. En raison du fait qu'un courant important peut être prélevé au moment de la charge ou du fonctionnement du relais, la sélection de "Extra Current" augmente la durée d'erreur de "Power" de ce port de 50 %.  Ce qui est important c'est que si ce port de connexion est court-circuité, l'appareil sera endommagé.

En cochant l'option "Dancing Light", les LED devant l'appareil s'allumeront de manière dansante.  Cette opération consiste à tester les entrées binaires "LED" saines de l'appareil.
