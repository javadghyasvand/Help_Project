Je nachdem, welche Funktionen mit dem Relais realisiert werden sollen, wird zum Zeitpunkt der Bestellung des Relais eine andere Hardwarekonfiguration festgelegt. Es besteht auch die Möglichkeit, die Hardware nach der Bestellung aufzurüsten, wenn Sie Funktionen entwickeln müssen. Um das Relais zu konfigurieren, muss die Relais-Hardware zunächst im Abschnitt Relay Hardware implementiert werden. In diesem Abschnitt werden eine Ansicht der Frontplatte des Relais und die Karten hinter dem Relais angezeigt.

Die Konfiguration von Hardware-Relais ist auf zwei Arten möglich:

Empfangen von Einstellungen vom Relais

Erstellen von Hardware durch Arbeiten mit Einheiten und Karten

Empfang der Hardware-Konfiguration vom Relais

Im Allgemeinen ist das Vebko-Relais in der Lage, den Typ (Kartentyp) und die Seriennummer der auf dem Relais platzierten Karten zu erkennen, aber die Platzierung der Karten sollte dem Relais über die Software mitgeteilt werden; falls erforderlich, kann diese Hardwarekonfiguration, die die Anzahl der Einheiten, den Typ und die Reihenfolge der Karten umfasst, vom Relais bezogen werden. Dazu muss die Software mit dem Relais verbunden sein. Dazu können Sie diese Einstellungen vom Relais erhalten, indem Sie auf das Symbol Hardware lesen in der Symbolleiste klicken oder die Option Hardware vom Relais lesen aus dem Menü Gerät auswählen.

Hinzufügen von Einheiten

Wenn Sie mit der rechten Maustaste auf den oberen Teil jeder Einheit klicken und die Option Einheit hinzufügen wählen, wird eine Einheit, die nur eine Futterkarte hat, nach der letzten Einheit hinzugefügt. Dies kann auch durch Klicken auf das Symbol "Add Unit" im oberen Bereich jeder Einheit geschehen.

Wenn Sie mit der rechten Maustaste auf jede Einheit klicken und "Insert empty unit in left" wählen, wird eine leere Einheit links von ihr eingefügt, wenn Sie "Insert empty unit in right" wählen, wird eine leere Einheit rechts von ihr eingefügt.

Entfernen einer Einheit

Um eine Einheit zu entfernen, klicken Sie mit der rechten Maustaste und wählen Sie Einheit entfernen oder klicken Sie auf das Symbol "Remove Unit" im oberen Teil der Einheit, um ein Symbol zu entfernen.

Die Einheit kann auch entfernt werden, indem Sie die gewünschte Einheit auswählen und die Löschtaste auf der Tastatur drücken.

Hinweis: Einheit 0 kann nicht gelöscht werden.

Hinweis: Beim Entfernen einer Einheit wird der Benutzer zunächst gefragt, ob ihre Signale im VFC verwendet werden. Bestätigt der Benutzer, werden die Signale auch im VFC gelöscht, und dann wird die Einheit gelöscht.

Kopieren einer Einheit

Um eine Einheit zu kopieren, muss die gewünschte Einheit zunächst im Speicher abgelegt werden, indem man sie mit der rechten Maustaste anklickt und die Option Einheit kopieren auswählt. Anschließend kann durch Rechtsklick auf jede Einheit und Auswahl der Option Einheit einfügen eine Kopie der Einheit auf der rechten Seite der ausgewählten Einheit erstellt werden.

Es ist auch möglich, die Tastenkombinationen Strg+X und Strg+V zu verwenden, anstatt die Optionen Einheit kopieren bzw. Einheit einfügen auszuwählen.

Hinweis: Eine Kopie der Einheit kann auf der rechten Seite hinzugefügt werden, indem Sie auf das Symbol "Duplizieren" klicken.

Um den Kopiervorgang zu vereinfachen, können Sie eine Kopie der Einheit in diesem Abschnitt erstellen, indem Sie die Strg-Taste drücken und die gewünschte Einheit in den blauen Abständen zwischen den Einheiten ziehen und ablegen.

Schneiden einer Einheit

Um eine Einheit auszuschneiden, muss die gewünschte Einheit zunächst im Speicher abgelegt werden, indem man sie mit der rechten Maustaste anklickt und die Option "Cut Unit" auswählt, dann mit der rechten Maustaste jede Einheit anklickt und die Option "Paste Unit" auswählt.

 

Es ist auch möglich, die Tastenkombinationen Strg+X und Strg+V zu verwenden, anstatt die Optionen Einheit ausschneiden und Einheit einfügen auszuwählen.

Um die Arbeit zu erleichtern, können Sie die gewünschte Einheit einfügen, indem Sie die Umschalttaste drücken, die gewünschte Einheit per Drag & Drop in die blauen Intervalle zwischen den Einheiten ziehen und die gewünschte Einheit in diesen Teil kleben.

Informationen zur Einheit

Wenn Sie auf eine Einheit doppelklicken oder mit der rechten Maustaste darauf klicken und die Option Einheiteneigenschaften auswählen, wird das Fenster Einheiteneigenschaften geöffnet und die Informationen über die Einheit, einschließlich der Beschriftungen, der Einheitennummern und der Beschriftungen ihrer Karten, werden angezeigt. In diesem Fenster können Sie die Gerätebezeichnung im Feld Bezeichnung ändern. Die Änderungen werden durch Klicken auf die Option OK gespeichert.

Die Bezeichnung kann geändert werden, indem Sie auf den Namen jeder Einheit im oberen Teil klicken.

Ändern des Kartentyps

Um eine neue Karte hinzuzufügen oder den Kartentyp zu ändern, klicken Sie mit der rechten Maustaste auf die Karte und platzieren Sie den Mauszeiger auf die Option "Change Card Type", um die Liste der angezeigten Karten zu sehen. Wenn Sie auf eine der Karten klicken, wird die ausgewählte Karte in diesem Teil hinzugefügt.

Hinweis: Jedes Relais kann nur eine CPU-Karte haben. Außerdem kann jede Einheit nur eine Einspeisekarte haben.

Entfernen einer Karte

       Um eine Karte zu löschen, klicken Sie mit der rechten Maustaste auf die gewünschte Karte und wählen Sie die Option "Remove Card".

Die Karte kann auch gelöscht werden, indem Sie die gewünschte Karte auswählen und die Löschtaste auf der Tastatur drücken.

Kopieren einer Karte

Um eine Karte zu kopieren, muss die Karte zunächst im Speicher abgelegt werden. Klicken Sie dann mit der rechten Maustaste auf die jeweilige Karte und wählen Sie die Option "Paste Card".

Anstatt die Optionen Karte kopieren und Karte einfügen auszuwählen, verwenden Sie die Tastenkombinationen Strg+C bzw. Strg+V.

Um die Arbeit zu erleichtern, können Sie gleichzeitig die Strg-Taste drücken und die gewünschte Karte per Drag & Drop in den Abschnitt ziehen, der sich auf andere Karten bezieht.Auf diese Weise wird eine Kopie der ausgewählten Karte erstellt.

Ausschneiden einer Karte

Um eine Karte auszuschneiden, muss sie zunächst im Speicher abgelegt werden. Klicken Sie mit der rechten Maustaste und wählen Sie die Option "Cut Card". Klicken Sie dann mit der rechten Maustaste auf jede Karte und wählen Sie die Option "Paste Card", um die Karte in diesen Bereich einzufügen.

Es ist auch möglich, die Tastenkombinationen Strg+X und Strg+V zu verwenden, anstatt die Optionen Ausschneiden bzw. Einfügen der Karte auszuwählen.

Um den Ausschneidevorgang zu vereinfachen, können Sie die Karte in diesem Abschnitt einfügen, indem Sie die Umschalttaste drücken und die Karte in den anderen Kartenabschnitt ziehen und ablegen.

Informationen zur Karte

Durch einen Doppelklick auf jede Einheit oder einen Rechtsklick auf die Einheit und die Auswahl der Option "Card Property" wird das Fenster "Card Properties" geöffnet. Auch die Signalinformationen dieser Karte sind in diesem Fenster verfügbar. In diesem Fenster können Sie die Bezeichnung im Feld "Card Label" ändern. Durch Klicken auf die Option "Save" werden die Änderungen gespeichert.

Hinweis: In diesem Fenster können Sie auch auf andere Karten in anderen Einheiten zugreifen. Dazu wird zunächst im Feld "Units" die gewünschte Einheit und dann im Feld "Unit Card" die Karte ausgewählt. Wenn Änderungen vorgenommen werden, wird der Benutzer gefragt, ob die aktuellen Karteneinstellungen gespeichert werden sollen; wenn Sie die Einstellungen speichern möchten, sollten Sie auf "Save" klicken.
