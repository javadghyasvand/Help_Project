DC Offset

 The transient performance specifications for some relays according to the IEC 60255-151 standard. In this post we are plan to outlines two key parameters related to transient performance: transient overreach and overshoot. Here's a bit more information about these terms:

Transient Overreach: This parameter is used to assess the effect of an offset waveform on the starting value accuracy of the relay element. For some relays, the transient overreach is specified to be 10% . This means that during instantaneous protection, the relay can tolerate a 10% overreach (or offset) in the value of the input signal. This overreach is relevant when assessing the relay's accuracy in determining when to operate.

Overshoot : Overshoot is defined as the difference between the operate time of the relay when the input energizing quantity is at a specified value and the maximum duration of the input energizing quantity. It pertains to the scenario when the input quantity is suddenly reduced (for overcurrent relay) or increased (for undercurrent relay) to a value below or above the relay's setting. If this change in input quantity is insufficient to trigger the relay's operation, it is considered overshoot. For instance, the overshoot time for the REF601 relay should be less than or equal to 100 milliseconds .

In summary, this text provides information about the transient performance requirements and standards, particularly focusing on transient overreach and overshoot, which are critical factors in assessing the reliability and accuracy of some relays in responding to electrical disturbances and faults.
