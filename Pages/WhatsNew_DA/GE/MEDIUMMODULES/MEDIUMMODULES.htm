MEDIUM MODULES   2021-07-16   21.07.16.01  

Sie können unter "Custom" einen Block mit dem spezifischen Namen "@Room" erstellen. Dieser Block ist für Module und geht beim Laden eines neuen xrio nicht verloren. Hinweis 1: Die Informationen in diesem Block werden mit anderen Informationen gespeichert und nicht in der xrio-Datei abgelegt. Er wird jedoch immer als letzter Block in "Custom" angezeigt und kann mit ihm verknüpft werden. Hinweis 2: Wenn in "VCC" ein externes Xrio importiert wird, befindet sich das Modul dieses Blocks weiterhin darin und wird in seinem Xrio angezeigt, aber das externe Xrio wird nicht geändert.

MEDIUM MODULES   2021-07-07   21.07.07.01  

Im Register "Extra Setting" erschien eine neue Option namens "Stop test after each shot test". Wenn diese Option aktiviert ist, werden nach dem Testen jedes Punktes im Schusstest zuerst die Relais vom Analogausgang im Prüfgerät ausgeschaltet und dann wird der nächste Punkt geprüft. Diese Option ist nicht in "AMT Sequencer" und nur in Modulen mit "Shot Test", "Search Test" und "Check Test" vorhanden. Dies geschieht z. B. für drei Prüfpunkte. Wie Sie sehen können, werden die LEDs des Abschnitts "Current Output A" für jeden Punkt eingeschaltet, wenn der Strom 10 Sekunden lang eingespeist wird, und sie erlöschen, wenn der Test jedes Punktes abgeschlossen ist. Das bedeutet, dass der Strom nicht injiziert wird und die entsprechenden Relais ausgeschaltet sind.

In "Load Relay Setting" wurde eine neue Option namens "Add missing blocks and parameters" erstellt. Es kann verwendet werden, um eine Vorlage aus "CSV"-Dateien von Relais zu erstellen. Dazu müssen Sie zunächst Relaismenüs in der Software anlegen. Mit dieser Methode werden Automatisch Blöcke und Parameter aus dem Relais erstellt. Es gibt zwei Möglichkeiten, Dateien im CSV-Format zu übertragen, entweder die Variablen haben einen Pfad oder es ist kein Pfad für sie definiert. Zunächst wird eine "CSV"-Datei vom "Schneider"-Relais geladen, die keinen definierten Pfad für ihre Blöcke und Parameter hat. In diesem Fall werden alle Variablen in einem Block namens "Missing Parameters" angelegt. Diesmal wird eine "CSV"-Datei aus dem "Toshiba"-Relais mit einem definierten Pfad für seine Blöcke und Parameter geladen. In diesem Fall, wenn ein Pfad für die Variablen vorhanden ist, werden sie nach demselben Standardpfad kategorisiert. Zwei Punkte sind wichtig; der eine ist, dass die "ID" jedes Blocks nach seinem Namen gemacht wird. Ein weiterer Punkt über Variablen ist, dass einige "IDs" verboten sind, so dass sie überprüft werden und wenn sie verboten sind, werden sie gültige "IDs". Daher kann sich die "ID" eines Parameters von seiner "ID" in der Datei unterscheiden, aber die erstellten "Foreign ID"s legen dasselbe mit der Datei fest. Der Typ aller Parameter wird als "String" ausgewählt, da sie in der Datei nicht angegeben sind.

Es wurde ein neues Feature zum Deaktivieren eines oder mehrerer Prüfpunkte, zum Deaktivieren von Prüfpunkten in einem oder mehreren Fehlertypen und zum Deaktivieren von Tests in einer oder mehreren Register erstellt, wenn ein Punkt oder eine Linie hinzugefügt wird. Um den Test beispielsweise an einem oder mehreren Prüfpunkten zu deaktivieren, wählen Sie sie zuerst aus, klicken dann mit der rechten Maustaste darauf und verwenden die Option "Active", um den Test an diesen Punkten zu deaktivieren. Wenn es einen Punkt oder eine Linie in Register gibt, können Sie den Test auch auf einer oder mehreren Register deaktivieren, indem Sie das neben jedem Register angelegte Kontrollkästchen aktivieren.

Medium Modules   2021-04-11   21.04.11.01  

Doppelte Optionen werden durch einen Rechtsklick auf die Prüftabelle eliminiert.

MEDIUM MODULES   2020-08-04   99.05.14.01  

Wenn Sie in der "Hardware Configuration" die Option "Show Actual Value" aktivieren, würden Sie nach dem Wechsel zu einer anderen Register und der Rückkehr zur "Hardware Configuration" sehen, dass diese Option deaktiviert ist und die Werte "False" sind.

In der neuen Version wird eine Funktion hinzugefügt, die einen "Threshold" für binäre Eingaben im Modus "Dry" definieren kann. Standardmäßig ist diese Zahl "1", aber der Benutzer kann in diesem Feld von "0" bis "2" Volt Gleichspannung eingeben. Zum Beispiel wird der Wert des binären "Threshold" auf "1" und "0.7" gesetzt. Dies bedeutet, dass, wenn die Spannung der binären "1" kleiner als 0.7 ist, wird der Binärkontakt geschlossen.

In allen Modulen, zum Beispiel dem "AMT Sequencer", stürzt es beim Ändern des "Scheme" nicht ab, wenn ein Modul oder eine Form vom Dock-Modus in den Abdock-Modus übernommen wird.

MEDIUM MODULES   2020-08-04   99.05.14.01  

In allen Modulen ist nach dem Öffnen der alten Dateien der Wert von "Threshold" bzw. die Schwellenspannung der Binäreingänge im Modus "Dry" auf 0.2 Volt eingestellt, was in der neuen Version korrigiert und durch einen Wert von 1 Volt ersetzt wird.

Wenn Sie in allen Modulen im Menü "File" die Option "Fill Excel File" auswählen, nachdem Sie eine Datei als Import geöffnet haben, wird standardmäßig eine Exportdatei dieselbe Importdatei festgelegt.

In der "Fill Excel File", nachdem Sie die Import- und Exportdateien ausgewählt haben und der Pfad klar ist, wurde dieser Pfad in den alten Versionen nicht gespeichert, aber in diesen Versionen gelöst.

In der "Fill Excel File" können Sie nach Eingabe wesentlicher Informationen durch Klicken auf "Start to Fill" die Optionen im Kontrollkästchen "Setting" bearbeiten. In dieser Version sind diese Optionen nach Auswahl der Option "Start to Fill" nur lesbar.

In der neuen Version wird das Problem "Fill Excel File" in "AMT Sequencer" im Abschnitt "Synchronizer" geändert.

Wenn in der Excel-Datei eine Seite namens vebko erstellt wird, möchten Sie Elemente auf dieser Seite ausfüllen, nachdem Sie die gewünschte Datei ausgewählt und in früheren Versionen auf die Option "Start to Fill" geklickt haben, erscheint eine Fehlermeldung.

MEDIUM MODULES   2020-11-03   99.08.13.01  

In mittleren Räumen wie "AMT Distance" wird nach dem Anschluss an das Gerät das Startsymbol zunächst deaktiviert und durch Hinzufügen von Punkten aktiviert.

In früheren Versionen dauerte es nach dem Drücken des Startsymbols eine Weile, bis der Test ausgeführt wurde, während in der neuen Version der Test ausgeführt wird, sobald Sie auf das Startsymbol klicken.

Einzelne Prüfprobleme werden in Schuss-, Such-, Kontrollprüfungen usw. behoben.

Alle Optionen, die in "Impedance View", "Differential Characteristic" usw. verfügbar sind, wie z. B. "Show all tact point", "Show z act point" usw. funktionieren korrekt.

Ebenfalls in diesem Abschnitt wurde die Option "Out of Range Area" erstellt, um den Bereich außerhalb des Einspritzbereichs des Geräts anzuzeigen.

In der "Characteristic View" werden die Tastenkombinationen wie "A", zur Anzeige von "Tact"-Punkten und "Z", zur Anzeige der Zonennummern der Punkte, "R" zur Anzeige der Zeilennummern der Punkte, "O" zur Bereich außerhalb des Einspritzbereichs des Geräts und "P", zur Anzeige des Istwerts der Impedanz sofort erstellt.

Rechtsklick-Optionen wie das Ändern der Hintergrundfarbe der "Characteristic View" werden geändert.

In allen mittleren Räumen werden durch Rechtsklick auf ein beliebiges Feld die Optionen "Link to Xrio", "Remove Link" und "Go to linked value" hinzugefügt.
