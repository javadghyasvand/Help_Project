Tester AMT105 Manual

1 : ERWEITERTE EINSTELLUNGEN VON „REPORT“, FOLGE 2

Im Abschnitt „Margin Setting“ können sie alle vier Margen des Textes festlegen. Im Abschnitt „Document Setting“ können Sie Seitenzahlen auf Berichtsseiten einfügen, indem Sie „Insert Page Number“ aktivieren. Durch Aktivieren von „Insert Page Breake before Main Heading“ wird außerdem jedes „HEADING“ des Berichts am Anfang jeder neuen Seite hinzugefügt.

Durch Auswahl von „Fit Width to Max Length Data“ im Abschnitt „Page Size“ wird die Größe der Tabellen im Bericht entsprechend der Seitengröße angepasst, während durch Auswahl von „Fix width“ die Seitengröße fixiert bleibt. Wenn die Tabelle größer als die Seite ist, ragen einige Teile der Tabelle über die Seitenbegrenzung hinaus.

In „Header/Footer Setting“ erscheint durch Ankreuzen von „Header“ das Feld „Header Text“, in das Sie den gewünschten Text eingeben können. Ebenfalls im Abschnitt „Header Height“ können Sie Ihre gewünschte Höhe für die „Header“ angeben.

Wenn Sie auf „Logo“ klicken und das Fenster „Import Logo for Report“ öffnen, können Sie ein Bild oder das Logo des Unternehmens in den „Header“ des Berichts einfügen.

Durch Aktivieren des Feldes „Footer Text“ wird das Feld „Footer Text“ angezeigt, in das Sie den gewünschten Text eingeben können. Auch im Abschnitt „Footer Height“ können Sie Ihre gewünschte Höhe für den „Footer“ angeben.

Durch Aktivieren von „Show Persons Sign“ und Auswählen von „Setting“ öffnet sich das Fenster „Persons Sign“, in dem Sie Ihrem Bericht bis zu vier Zeichen hinzufügen können. Wenn Sie auf „Add Persons Sign“ und auf „Signature“ klicken und „Signature“ auswählen, können Sie Ihrem Bericht eine Unterschrift hinzufügen. In den Abschnitten „Title“ und „Value“ wird jeweils der Titel und der Name der Person eingetragen.

Im Abschnitt „Title“ des Abschnitts „Setting“ können Sie einen allgemeinen Titel für alle vier Signaturen eingeben und ein Kästchen für jede Signatur hinzufügen, indem Sie „Show Border in Report“ aktivieren. Außerdem werden im Abschnitt „Repeat Mode“ durch Auswahl „Show in all Pages“ die eingegebenen Unterschriften auf allen Seiten angezeigt. Wenn Sie jedoch „Show in Last Page“ auswählen, werden diese Signaturen nur auf der letzten Seite angezeigt.

Im Abschnitt „Extra Setting“ und in den Feldern „Tested by“, „Approved by“, „Company“ und „Comment“ können Sie den Namen des Testteilnehmers, des Vorgesetzten und der Firma sowie Kommentare eingeben. Beachten Sie, dass Sie ihre Optionen im Strukturdiagramm auch unter dem abstrakten Zweig überprüfen müssen, um diese Elemente im Ausgabebericht anzuzeigen, außer deren Optionen in diesem Abschnitt zu überprüfen.

Wenn Sie „Browse Custom“ wählen, können Sie bestimmte Informationen der Datei „Xrio“ im Bericht anzeigen, indem Sie die entsprechende Option markieren. Um beispielsweise „Name Plate“ und „Location“ anzuzeigen, sollten Sie diese Optionen im Baumdiagramm „Test Objekt“ des Abschnitts „Device“ auf der Seite „Browse Custom“ und „Xrio Block Report Setting“ überprüfen, damit die beabsichtigten Informationen speziell im Bericht angezeigt werden.
